# Linux Packages

Install these packages on owned Linux hosts before running `./linux/bootstrap`:

- `zsh`
- `vim`
- `git`
- `git-delta`
- `eza`
- `fzf`
- `mise`
- `ripgrep`
- `zoxide`
- `zsh-autosuggestions`
- `zsh-syntax-highlighting`

On Debian or Ubuntu hosts, the zsh config expects the packaged completion and plugin files under `/usr/share`.
